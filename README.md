# Smart Bookmark App

## Features
- Google Authentication using Supabase
- Add & delete bookmarks
- User-specific private bookmarks
- Real-time updates using Supabase Realtime
- Deployed on Vercel

## Tech Stack
- Next.js (App Router)
- Supabase (Auth, Database, Realtime)
- Tailwind CSS
- Vercel

## Challenges & Solutions
1. Google OAuth redirect issues  
   → Solved by configuring redirect URLs in Supabase.

2. Real-time updates not working  
   → Implemented Supabase Realtime channels.

3. Data security between users  
   → Implemented Row Level Security (RLS) policies.
